Setup Instructions:
1. Install and open XAMPP.
2. Start Apache and MySQL.
3. Copy the blood_donation folder into C:\xampp\htdocs\
4. Open phpMyAdmin in the browser:
   http://localhost/phpmyadmin
5. Create/import the database using the SQL file:
   blood_donation_db.sql
6. Open the system in the browser:
   http://localhost/blood_donation/
